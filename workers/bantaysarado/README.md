# bantaysarado — Cloudflare Workflows Starter

A real-time, interactive demonstration of [Cloudflare Workflows](https://developers.cloudflare.com/workflows) with live updates via WebSockets and Durable Objects.

## Getting Started

```bash
npm install
npm run dev
```

Visit `http://localhost:5173` to see the interactive demo.

## Deployment

```bash
npm run deploy
```
